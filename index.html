Textos

Utilize tags adequadas para títulos, subtítulos, parágrafos e listas no seu conteúdo textual.
O leitor de telas vai identificar cada elemento para o usuário, permitindo ainda navegar de título em título
(o que é bastante útil para portais de notícias, por exemplo).


Imagens

No HTML, as imagens devem possuir o atributo alt com um texto alternativo que descreva seu conteúdo.
Vale sinalizar que todos os elementos visuais ou multimídia, como ícones, vídeos e áudios, precisam de um
elemento textual que os descrevam.



Links

Links sublinhados são úteis para usuários com dificuldade para discernir contraste.
Textos de links devem ser descritivos (e não apenas “clique aqui”). Como por exemplo:


MODO ERRADO

Para saber acessar o YouTube clique aqui.


MODO CERTO

YouTube

Cores

Pense no constrate em diversos fundos e não apenas no branco.
Garanta o contraste entre fundo e texto em diversos modos de exibição.
Usar cores intuitivas.


Tabelas

Tabelas devem ser escritas em HTML com células de cabeçalho associadas
corretamente às suas colunas ou linhas, para uma leitura mais clara do leitor de telas.
